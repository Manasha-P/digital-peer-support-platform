# Digital Peer Support Platform - Backend

Professional MERN stack backend with userType-based topic filtering for structured peer support.

## 🚀 Features

✅ **Role-Based Authentication** - User, Peer Supporter, Admin
✅ **UserType System** - College Student, School Student, Employee  
✅ **Dynamic Topic Filtering** - Topics filtered based on userType
✅ **JWT Authentication** - Secure token-based auth
✅ **Session Management** - Book, track, and complete support sessions
✅ **Real-Time Chat** - Socket.io integration (ready)
✅ **Notification System** - In-app notifications
✅ **Feedback & Rating** - Rate peer supporters after sessions
✅ **Admin Approval** - Approve/reject peer supporter applications

## 📋 Prerequisites

- Node.js (v16+)
- MongoDB (local or cloud)
- npm or yarn
